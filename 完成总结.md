### 方式 2：后台运行

```bash
ssh 3090GPU2
cd /home/scale/kangjay/kaohe
nohup python -m src.ui.app > app.log 2>&1 &
```

查看日志：
```bash
tail -f /home/scale/kangjay/kaohe/app.log
```

停止服务：
```bash
pkill -f "src.ui.app"
```
